# 🚚 Safe Move – Move With Safety

Welcome to **Safe Move**, a sleek and responsive frontend project built using **Bootstrap 5**. This project is designed as a modern landing page for a moving service company that prioritizes **safety, reliability, and simplicity**.

---

## 🌟 Overview

Safe Move is a clean and user-friendly website UI that showcases a fictional packers & movers service. It demonstrates how Bootstrap can be used to rapidly build beautiful, responsive layouts with minimal effort.

Whether you're learning frontend development or building your own startup landing page — this project is a great starting point.

---

## 🎯 Features

✨ **Responsive Navbar**  
- Collapsible menu for mobile devices  
- Dropdown services section  
- Search bar for postcode checking  

🎉 **Hero Section**  
- Eye-catching headline: *"Move With Safety"*  
- Call-to-action buttons (Get a Quote / Contact Us)  
- Clean layout with image support  

💼 **Why Choose Us Section**  
- Highlights key strengths:
  - Professional service  
  - Countrywide coverage  
  - Personal touch  
- Icon-based feature cards  

📸 **Image Carousel**  
- Smooth sliding carousel  
- Perfect for showcasing happy customers or services  

🎨 **Custom Styling**  
- Soft beige background  
- Blue & wheat themed buttons  
- Hover effects for interactive feel
